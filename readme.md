## 🚀 Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/ChiragSavani1/unit-testing-app.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run tests:
   ```bash
   npm test
   ```
4. Start dev server:
   ```bash
   npm start
   ```

## 🔧 Tech Stack
- Jest (testing)
- Express (server)
- HTML/CSS (UI)
